## Getting started

```bash
git clone https://github.com/Mrudula-itsjuzme/our-code.git
cd our-code

npm install
npm start
```

Open `http://localhost:3000` in your browser.

---

## Available scripts

```bash
npm start      # run the development server
npm test       # run tests
npm run build  # create a production build
```
